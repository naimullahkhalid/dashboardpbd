💡 Panduan Format Excel / CSV:

  • Pastikan lajur wajib wujud: Nama, MyKID (atau Pengenalan).
  • Format MyKID di Excel MESTI ditukar format Number (0 titik perpuluhan) sebelum save as CSV untuk elak ralat (cth `1.41E+11`).

  • Markah mestilah mengandungi nombor TP (contoh: "TP3", "TP 4", atau "4").
